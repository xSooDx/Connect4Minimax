To run on Windows, run 'Connect4.exe'

Use Code Blocks to open and build the project.

Class header files are in '/include'
Class implementation files are in '/src'
ALl object files are in '/obj'
The main file is in the root directory.

If you build the project using Code Blocks. A new .exe is genereated in '/bin'

'Connect4.cbp.mak' is the makefile for the project.
